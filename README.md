# Superpowers for Robotics

A specialized fork of [Superpowers](https://github.com/obra/superpowers) with domain-specific skills for robotics development, integrated with [Yato](https://github.com/RedEarth-Robotics/yato) token optimization.

## Local Installation

Clone this repository to access the robotics skills:

```bash
git clone https://github.com/RedEarth-Robotics/robotics-superpowers.git
```

The skills are located in the `skills/` directory. Load them alongside your main Superpowers installation by adding the `skills/` path to your harness's skill discovery configuration.

**Prerequisite:** Install the original [Superpowers](https://github.com/obra/superpowers) first.
